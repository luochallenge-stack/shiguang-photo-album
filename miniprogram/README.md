## 当前范围

- 站内账号注册、登录与退出
- 普通成员只读访问
- 全部影像与文件夹浏览
- 加密文件夹密码解锁
- 管理员选择文件夹后可拍照或从相册批量上传图片；单个上传任务最多 50 张，每轮选择最多 9 张
- 图片预览和视频播放
- 与网页端共用用户、权限、日志、数据库和云存储

视频上传、删除、重命名、文件夹管理和用户授权继续使用网页管理端完成。
